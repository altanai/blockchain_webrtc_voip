# SMart Contract using web3.js

## web3.js - Ethereum JavaScript API

JS API which connects to the Generic JSON-RPC spec.

Via nodejs 
```shell script
npm install web3
```

or VIA CDN 
```html
<script src="https://cdn.jsdelivr.net/npm/web3@latest/dist/web3.min.js"></script>
```
